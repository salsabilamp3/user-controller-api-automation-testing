1. Untuk melihat hasil generate test report buka file surefire-report.html menggunakan browser
2. Terdapat 3 bagian pada halaman ini yaitu summary, package list dan test cases. Lalu pada bagian bawah terdapat rangkuman dari semua failures yang ada.
3. Pada bagian summary terdapat rangkuman dari keseluruhan test
4. Pada bagian package list terdapat list dari test yang ada yaitu untuk keempat fitur, terlihat jumlah test, failures dan lainnya.
5. Selanjutnya pada test cases berisi detail dari semua test yang dilakukan, jika test cases berhasil maka icon yang muncul adalah ceklis. Jika test case fail maka akan muncul icon x dan juga pesan kesalahan untuk menunjukan actual result serta expected nya.
